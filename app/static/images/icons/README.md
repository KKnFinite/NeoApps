# NeoApps Hex Icons

This folder holds locked hex app and NeoNode icons for cards, launchers,
favicons, and PWA install manifests.

The simple two-letter PNG files in this folder are the source artwork. Generated
install sizes live in per-app/per-node subfolders and should be regenerated from
these source PNGs when the locked artwork changes.

Current source abbreviations:

- `na.png` -> NeoApps / NeoPortal install identity
- `ng.png` -> NeoGateway
- `mb.png` -> NeoMotherBrain
- `er.png` -> NeoErmac
- `sk.png` -> NeoSektor
- `sc.png` -> NeoScorpion
- `rp.png` -> NeoReptile
- `sz.png` -> NeoSubZero
- `rn.png` -> NeoRain
