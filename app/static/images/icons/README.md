# NeoApps Icon Placeholders

This folder holds app and NeoNode icon placeholders for favicon/PWA work.

The root site icon is NeoPortal (`NP`). NeoGateway (`NG`) remains separate and
keeps its own existing logos/assets.

Current placeholder abbreviations:

- `neoportal` -> NP
- `neogateway` -> NG
- `neostaffing` -> NS
- `neobid` -> NB
- `motherbrain` -> MB
- `sektor` -> SK
- `ermac` -> ER
- `scorpion` -> SC
- `reptile` -> RP
- `subzero` -> SZ
- `rain` -> RN

These are temporary generated PNG assets. Final app/node artwork can replace the
files in-place when available.
